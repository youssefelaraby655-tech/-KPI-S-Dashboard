# KPI's Dashboard - Sales Performance Analysis

## Project Overview
This project provides a comprehensive analysis of sales performance using Power BI. It focuses on tracking key performance indicators (KPIs) to drive data-driven decisions.

## Key Insights
* **Total Sales:** Analysis of revenue trends over time.
* **Profitability:** Detailed look at profit margins across different segments.
* **Customer Behavior:** Insights into top-performing products and customer demographics.

## Tools Used
* **Power BI:** For data visualization and DAX modeling.
* **Excel:** For data cleaning and preparation.
